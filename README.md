# 🚢 SIGenie.ai
- AI를 활용한 해운 산업의 선적 지시서 처리 자동화 시스템
- **SI**(Shipping Instruction, 선적 지시서) + **Genie**
- LLM, RAG, and multi AI Agent

## WorkFlow
### **chapter 1**

1. 예약 정보 불러오기
2. 선적 지시서 불러오기
3. 서류 완전성 확인
4. 요약 보고서 생성

### **chapter 2**

1. 수출자 및 수입자 확인
2. 회사 정책 준수 검증
3. 실시간 항구, 선박 뉴스 검색
4. 검증 보고서 생성
